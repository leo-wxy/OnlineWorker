Configure Telegram notifications

Telegram notifications need a Bot Token and a recipient user ID. After saving and enabling the channel, OnlineWorker sends concise updates when a task completes or needs attention.

Get a Bot Token

  1. Search for BotFather in Telegram.
  2. Send /newbot to create a bot.
  3. Copy the token returned by BotFather into Bot Token.

Get the recipient user ID

  1. Send any message to the bot you created.
  2. Use your Telegram user ID as Recipient User ID.
  3. Save the configuration and enable the Telegram channel.

Notes

  • A bot cannot start a private chat with a user who has never messaged it.
  • This is the dedicated notification channel and does not change the main Telegram bot group setup.
