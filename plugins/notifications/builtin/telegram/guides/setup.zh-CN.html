配置 Telegram 通知

Telegram 通知需要一个 Bot Token 和一个接收用户 ID。保存后可以启用渠道，任务完成或需要处理时会发送简短通知。

获取 Bot Token

  1. 在 Telegram 搜索 BotFather。
  2. 发送 /newbot 创建机器人。
  3. 复制 BotFather 返回的 token，填入 Bot Token。

获取接收用户 ID

  1. 先给你创建的机器人发送任意一条消息。
  2. 使用你自己的 Telegram 用户 ID 作为 接收用户 ID。
  3. 保存配置并启用 Telegram 渠道。

注意

  • 机器人不能主动给从未私聊过它的用户发消息。
  • 这里配置的是通知专用渠道，不影响主 Telegram Bot 的群组交互配置。
