# 10-03: Extract Python Workspace Pure Helpers

## Objective

Reduce `bot/handlers/workspace.py` by moving pure topic, callback token, and history-formatting helpers into a dedicated module while preserving all Telegram workflow behavior.

## Scope

- Extract workspace topic label/name helpers.
- Extract thread-open token and workspace callback identity helpers.
- Extract history turn timestamp normalization, signature generation, Telegram message formatting, and sync batch construction.
- Keep Telegram callback handlers, provider hook calls, thread lookup, topic creation, and storage writes inside `workspace.py`.
- Keep compatibility aliases in `workspace.py` so existing tests and imports do not break.

## Non-Goals

- No changes to Telegram callback data formats.
- No changes to topic creation, topic rename, or history replay behavior.
- No changes to provider registry hooks or local thread discovery.
- No cleanup of broader workspace orchestration in this slice.
