# Sketch 001: Task Board Attention Layout

## Design Question

How should the existing Task Board Tab evolve into a focused pending-action center without losing its running and followed Session overview?

## How to View

Open `.planning/sketches/001-task-board-attention-layout/index.html`.

## Variants

- **A: 上下分层** — A compact actionable queue spans the top; running and followed Sessions remain visible below.
- **B: 收件箱工作台** — A narrow pending-item inbox drives a detailed action and activity pane.
- **C: 统一时间流** — Pending, running, and followed items share one chronological operational feed with lightweight filters.

## What to Look For

- How quickly the highest-priority pending item is discoverable.
- Whether pending actions have enough context to act safely.
- Whether running Sessions remain easy to scan and interrupt.
- Whether the layout stays useful with many items and without creating local-only state.
